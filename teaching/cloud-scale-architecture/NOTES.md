# Notes

- 언어: 레슨 본문은 한국어, 기술 용어는 영어 원어 유지 (사용자 평소 대화 패턴).
- 사용자는 이 시스템의 설계자 — 기초 개념(티켓, 워커, PR 플로우)은 건너뛰고
  스케일 설계의 "왜"에 집중. ZPD는 "자기 시스템은 알지만 분산 인프라
  프리미티브(lease/fencing, submit queue, microVM snapshot)는 처음 소화".
- 레슨은 짧게 — 한 레슨 한 개념, 퀴즈로 retrieval practice.
- 퀴즈 선택지는 길이 균등 (포맷으로 정답 힌트 주지 않기).
- 스펙 자체가 한 번 외부 리뷰로 교정된 이력 있음 — 교정에서 살아남은
  숫자만 가르칠 것 (예: Temporal 상단 $39k pre-discount, Modal ~$104k).
